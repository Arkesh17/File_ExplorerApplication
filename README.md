# File_ExplorerApplication
Devloped a console-based file explorer application in C++ that interfaces with linux operating system to manage files and directories
This project, File Explorer using C++, provides a command-line interface that allows users to perform file operations such as creating, deleting, copying, moving, searching, and viewing permissions. The application directly uses UNIX system calls such as open(), stat(), chmod(), opendir(), and readdir() for low-level file handling.
